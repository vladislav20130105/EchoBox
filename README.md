# 🎵 EchoBox

Простая и удобная библиотека звуков с поддержкой MP3 и WAV файлов.

## ✨ Возможности

- 🎵 **Поддержка форматов**: WAV, MP3, OGG, FLAC, AAC, M4A
- ▶️ **Воспроизведение**: запуск выбранного звука
- ⏹️ **Пауза**: остановка с сохранением позиции
- 🛑 **Стоп**: полное завершение воспроизведения
- ➕ **Добавление**: простое добавление файлов в библиотеку
- 📝 **Переименование**: изменение названий звуков
- 🗑️ **Удаление**: удаление звуков из библиотеки
- 🔢 **Горячие клавиши**: цифровые клавиши для быстрого доступа
- 🔍 **Поиск**: мгновенный поиск по названиям и форматам
- 🎵 **Плейлисты**: создание и управление плейлистами
- 🔂 **Режим плейлиста**: переключение между библиотекой и плейлистами
- 📊 **Прогресс-бар**: визуализация воспроизведения с перемоткой
- 🎯 **Компактный интерфейс**: максимум песен на экране

## 🚀 Установка

### Из исходников:
```bash
git clone https://github.com/your-username/echobox.git
cd echobox
pip install -r requirements.txt
python echobox.py
```

### Через pip (когда будет опубликовано):
```bash
pip install echobox
```

## 🎮 Использование

1. **Запустите приложение**
2. **Нажмите ➕ для добавления звуков**
3. **Выберите звук из списка**
4. **Используйте кнопки управления**:
   - ▶️ Воспроизведение
   - ⏹️ Пауза
   - 🛑 Стоп
5. **Используйте поиск** для быстрого нахождения звуков
6. **Создавайте плейлисты** для организации музыки
7. **Используйте прогресс-бар** для перемотки

## 🎵 Плейлисты

- **Создание плейлистов** через кнопку "➕ Новый"
- **Добавление звуков** через контекстное меню (правый клик)
- **Переключение режимов** между библиотекой и плейлистами
- **Автоматическое добавление** в плейлист при режиме плейлиста

## � Прогресс-бар и перемотка

- **Визуализация** текущей позиции воспроизведения
- **Перемотка** - кликните по прогресс-бару для перемотки
- **Отображение времени** - текущее / общее время
- **Сохранение позиции** при остановке
